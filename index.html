Yippi!

Yippi is a e621 API wrapper for Python 3, it is easy to use, fast, and doesn't require any extra dependencies!

Read the Docs

EASY

Yippi is pretty easy to use, with only a few lines, you already get the data from e621! Isn't that great? Even you're new to Python, or just upgraded to Python 3.x, you'll find this really easy to play with!

FAST

Using the prebuilt urllib package, it plays really fast because Python itself already familiar with the dependencies! The only thing matters is your internet speed, of course.

OPEN SOURCE

Yippi is open source, and it's available at GitHub! So you don't need to worry about the module stealing your data or anything like that, you could also contribute to the project! I would be glad if you're interested in contributing this project!
